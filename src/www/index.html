What is Otter?

Otter is a visual tool to quickly analyze iOS applications through data from the iOS App Store.

At a glance:

  • Analyze user sentiment using data from the last 100 user reviews.
  • Compare sentiment across multiple applications simulatenously.
  • Compare applications by sentiment and popularity.
  • The most commonly used words are represented as circles - size corresponds to frequency and color to sentiment. The larger a circle, the more commonly used the word. The greener the circle, the better a word's sentiment.

This is a working prototype and has limited support for non-US applications.
